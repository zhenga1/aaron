Introduction:

In 1D1D1D and 2D2D2D calculus, a common problem that was encountered was solving for the exact area bounded by curves. For example, we may want to find the area bounded by the yyy-axis and the curve, sketched below: y=−x2+5y=-x^2+5y=−x2+5 ![[chrome_gOy0K3fqYL.png]] The idea is that such areas will be computed as the sum of the areas of infinite rectangles, each of height y=−x2+5y = -x^2+5y=−x2+5 and an infinitesimal width denoted by dxdxdx. ![[Pasted image 20221025161037.png]] (https://jmahaffy.sdsu.edu/courses/f00/math122/lectures/riemann_sums/images/i_int40.gif)

As shown in the above image, the sum of the green areas is an approximate answer for the actual area under the red curve. To find the exact area in 1D1D1D calculus, we compute the integral of the function for which we want to find areas for, which is expressed as: ∫f(x)dx,Eq1\int f(x)dx, Eq1∫f(x)dx,Eq1 for some function y=f(x)y=f(x)y=f(x). The above expression represents an indefinite integral, as the bounds of integration are not defined. In general, the expression: ∫abf(x)dx,Eq2\int_a^b f(x)dx, Eq2∫ab​f(x)dx,Eq2 Represents a definite integral (since bounds are given), and gives the area below the function y=f(x)y=f(x)y=f(x) as xxx varies from (a,b)(a,b)(a,b) in the x−yx-yx−y plane.

The ability to find such areas has many applications. For example, to finding the total work done by a motor in moving a certain object, we can draw a graph of Force over time(as shown below). We have that the work W=F×dW=F\times dW=F×d , which is the product of distance by Force.

![[Pasted image 20221025155319.png]] (https://pediaa.com/wp-content/uploads/2015/07/How-to-Calculate-Work-Done_Force_vs_Distance_Graph.jpg) If we assume a force by distance graph as shown above, then we see that the area above is simply the expression: ∫x1x2F(x)dx\int_{x_1}^{x_2} F(x) dx∫x1​x2​​F(x)dx

For 1D1D1D-integrals of a function f(x)f(x)f(x), the indefinite integral is equivalent to a function F(x)F(x)F(x) which has the unique property that F′(x)=f(x)+cF'(x) = f(x) + cF′(x)=f(x)+c, where c∈Rc\in Rc∈R is some constant.

This unique property is known as the fundamental theorem of calculus. By extension, any integral expression represented by Eq2 above can be written as: ∫abf(x)dx=F(b)−F(a)\int_a^b f(x)dx = F(b)-F(a)∫ab​f(x)dx=F(b)−F(a), since the constant terms would cancel out.

Double Integrals

After laying the framework for solving for exact areas within 1D1D1D functions, we now attempt to extend this idea to 2D2D2D functions.

In essence, an integral of a 1D1D1D function is an integral that solves for the exact area underneath a function y=f(x)y=f(x)y=f(x) across the range (a,b)(a,b)(a,b) .The precise bounds of integration can be denoted using only two values, the upper and lower bounds. This is because since the input space of f(x)f(x)f(x) is the x-axis, the input space is a line. A subpart of a line is a line segment, which can be precisely defined by two points.

However, we see that 2D2D2D functions, unlike 1D1D1D functions, take on 222 unknown variables. Hence, rather than a line, the input space of 2D2D2D functions is a plane, defined by two orthonormal axes that each represent 111 unknown. Unlike the 1D1D1D bounds of integration, the 2D2D2D bounds of integration cannot be defined by just two values. In fact, an infinite amount is needed, since any closed contour shape on a plane can effectively be the bounds of integration for a 2D2D2D function.

![[Pasted image 20221025162811.png]] (https://i.stack.imgur.com/yn8Ux.png)

Above contours are all valid bounds of integration. As a result, we represent the contour usually using the symbol DDD, rather than attempt to use specific values to define it.

Unlike the 1D1D1D version, the integral of a 2D2D2D-function z=f(x,y)z=f(x,y)z=f(x,y) gives the volume, not the area, of the 3D3D3D solid that lies below the function f(x,y)f(x,y)f(x,y).

We can visualise the double integral of a given 2D2D2D function z=f(x,y)z=f(x,y)z=f(x,y) as the sum of an infinite amount of volumes of rectangular prisms with their bases lying on the bounds of integration, which each have height given by f(x,y)f(x,y)f(x,y) and an infinitesimal area of the base, which has length dxdxdx and width dydydy. ![[Acrobat_hP0loigSWG.png]] (Calculus Early Trancendentals, James Stewart)

As seen above, each of the heights of the rectangular prisms is given by z=f(x,y)z=f(x,y)z=f(x,y), whilst the area of the bases is given by dxdydx dydxdy. Just like the 1D1D1D equivalent, the exact volume of the body below a 2D function is given whenever dx,dydx,dydx,dy approches zero, and the number of rectangular prisms we draw approach infinity. As there are more rectangular prisms, the volume we get becomes more accurate. This phenomenon is further illustrated below: ![[Acrobat_3SdheLuLon.png]] (Calculus Early Trancendentals, James Stewart)

To denote the exact volume, we can write the double integral.

The double integral of a function f(x,y)f(x,y)f(x,y) over the rectangle R=(a,b)×(c,d)R = (a,b)\times (c,d)R=(a,b)×(c,d) is represented by ∫R∫f(x,y)dA=lim⁡m,n→∞Σi=1mΣj=1nf(xij,yij)ΔA\int_R\int f(x,y)dA = \lim_{m,n\rightarrow \infty}\Sigma_{i=1}^m\Sigma_{j=1}^nf(x_{ij},y_{ij})\Delta A∫R​∫f(x,y)dA=limm,n→∞​Σi=1m​Σj=1n​f(xij​,yij​)ΔA Where xijx_{ij}xij​ ranges from (a,c)(a,c)(a,c) in mmm fixed intervals of length c−am\frac{c-a}{m}mc−a​ , while yijy_{ij}yij​ ranges from (b,d)(b,d)(b,d) in nnn fixed intervals of length c−an\frac{c-a}{n}nc−a​.

To evaluate a double integral for the function f(x,y)f(x,y)f(x,y) over a rectangular region DDD, where xxx is bounded by (a,c)(a,c)(a,c) and yyy is bounded by (b,d)(b,d)(b,d), we can use Fubini's theorem, which states: ∫D∫f(x,y)=∫ac∫bdf(x,y)dydx=∫bd∫acf(x,y)dxdy\int_D\int f(x,y) = \int_a^c\int_b^d f(x,y)dydx = \int_b^d\int_a^c f(x,y)dxdy∫D​∫f(x,y)=∫ac​∫bd​f(x,y)dydx=∫bd​∫ac​f(x,y)dxdy If the integral exists.

We call a function integrable if the above limit always exists and always equals to some value LLL, regardless of which sets of points (xij,yij)(x_{ij},y_{ij})(xij​,yij​) we choose along the m×nm\times nm×n fixed intervals. See the definition for limits in the Chapter 14 essay.

Application of Double Integrals

In 1D1D1D calculus, the ability to solve for exact areas under a 1D1D1D curve proved invaluable. With this ability, we were able to accurately calculate many values based on other values, such as the exact distance s at given time ttt based on velocity or even acceleration over time ttt graphs. Alternatively, we were also able to calculate solutions to finding the exact work W done by certain objects over a certain distance, when applying a variable force FFF.

Similar to its 1D1D1D counterpart, double integrals for 2D2D2D functions allow us to solve for many different quantities. In the bottom I will list 333 examples.

But f(x,y)=1f(x,y)=1f(x,y)=1 for all points, so essentially this is just the volume of the prism solid with DDD as base, and extruded a height of 111. Hence,

Change of Variables in Multiple Integrals

It is all and well that integrals have many real-life applications. However, sometimes the computation of certain integrals have challenges. In some cases the bounds of integration is arbitrary and hard to calculate. For 1D1D1D calculus, we change bounds of integration through substitution as follows: ∫abf(x)dx=∫cdf(g(u))g′(u)du\int_a^b f(x)dx = \int_c^d f(g(u))g'(u)du∫ab​f(x)dx=∫cd​f(g(u))g′(u)du where (c,d)(c,d)(c,d) is the corresponding bounds such that (g(c),g(d))=(a,b)(g(c),g(d))=(a,b)(g(c),g(d))=(a,b)

On occasion, the change of variables framework is also very useful in multivariable functions.
