Invertibility of Linear Transformations

  • Theorem: A is invertible, if and only if(iff) the columsn of A are lineraly independent
      1. If columns of AAA are lin. dep. then A−1A^-1A−1 does not exist
      1. If A−1A^{-1}A−1 exists, then the cols of A are linearly independent
  • Proof concept: Assume linear dependene and invertibility and show that it is a contradiction

From linear dependence: we have that if A has linearly dependent columns, then there must exist a⃗≠0⃗\vec{a}\neq\vec{0}a=0
such that: A×a=0⃗A\times a = \vec{0}A×a=0

Assume A−1A^{-1}A−1 exists A×a=0A\times a = 0A×a=0 Assume A−1A^{-1}A−1 exists then we must have A−1×A×a=A−1×0⃗A^{-1} \times A\times a = A^{-1}\times \vec{0}A−1×A×a=A−1×0

We have, given a matrix A A=[abcd]A = \begin{bmatrix} a & b\\c & d\end{bmatrix}A=[ac​bd​] the determinant is given by a∗d−b∗ca*d-b*ca∗d−b∗c

Equivalent Statements:

  • Matrix A is invertible
  • A∗x=bA*x = bA∗x=b has a unique solution
  • A has linearly independent columsn(A is full rank)
  • A has trivial nullspace

Today and next time's Jargon:

Vector space:

A vector space V is set of vectors made up of scalars and two operators * *, + that satisfy, given x⃗∈V\vec{x} \in Vx∈V

  • a×xa\times xa×x is part of V
  • x+yx+yx+y is part of V
  • x+(y+z)=(x+y)+zx + (y+z) = (x+y) + zx+(y+z)=(x+y)+z (associativity)
  • x+y=y+xx+y = y+xx+y=y+x (commutativity)
  • ∃0∈V\exists0 \in V∃0∈V s.t x+0⃗=xx + \vec{0} = xx+0=x (additive identity)
  • ∃(−x)∈V\exists (-x) \in V∃(−x)∈V s.t. x+(−x)=0x+(-x)=0x+(−x)=0 (additive inverse)
  • a∗(x+y)=a∗x+a∗ya*(x+y) = a*x+a*ya∗(x+y)=a∗x+a∗y ( distributivity)
  • a×(βx)=(αβ)×xa\times(\beta x)=(\alpha \beta)\times xa×(βx)=(αβ)×x
  • (α+β)×x=αx+βx(\alpha + \beta)\times x = \alpha x + \beta x(α+β)×x=αx+βx
  • 1×x=x1\times x= x1×x=x
