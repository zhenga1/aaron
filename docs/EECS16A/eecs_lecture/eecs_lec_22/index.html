Vectors can be thought of as signals We can define literally anything as a signal, such as a vector representing the faces of Ana and Miki.

Moving example is GPS, try to find the distances with the saelites.

  • 24 satelites
    • Known position
    • Time synchronised
    • 8 usually visible
  • Problem:
    • Classify which satelite is transmitting
    • Estimate distance to GPS
    • Estimate position from noisy data
  • Tools:
    • Inner product
    • Cross correlation
    • Least Squares

Localization

  • Satelites transmit a unique code
    • Radio signal
  • Signal is received and digitized by a receiver

  1. Interference
  2. Timing
